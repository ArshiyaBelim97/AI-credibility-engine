#AI-Credibility-Engine

##Overview

###AI-Credibility-Engine is a full-stack application designed to analyze text, images, audio, and videos to detect whether the content is AI-generated or authentic, providing a credibility score for each input.
It combines FastAPI for the backend, React for the frontend, and state-of-the-art Hugging Face Transformers models for content detection.

##Features
###✅ Detect AI-generated text.
###✅ Detect AI-generated images.
###✅ Detect AI-generated audio.
###✅ Detect AI-generated videos.
###✅ Web-based interactive frontend for uploading files and viewing results.
###✅ Credibility score output for each analysis.

##Tech Stack
###Backend: Python, FastAPI, Uvicorn, Transformers, Torch
###Frontend: React, Tailwind CSS
###Machine Learning: Hugging Face Transformers, PyTorch
###Deployment: Local development (can be containerized using Docker)
